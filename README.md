# Resume Analyzer (C++)

A console-based Resume Analyzer built using Object-Oriented Programming and Data Structures in C++.

## Features

- Resume Parsing
- Job Description Parsing
- Skill Matching
- ATS Score Calculation
- Report Generation
- Multiple Resume Support

## Concepts Used

- Object-Oriented Programming
- File Handling
- STL Containers
- Data Structures
- String Processing
- Modular Design

## Project Structure

data/      -> resumes and job descriptions
inc/       -> header files
src/       -> implementation files
output/    -> generated reports

## How to Run

Compile:

```bash
g++ src/*.cpp -Iinc -o ResumeAnalyzer
```

Run:

```bash
ResumeAnalyzer.exe
```
